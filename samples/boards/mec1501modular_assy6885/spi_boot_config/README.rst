MEC15xx MEC15xx modular board sample SAF demo
#############################################

Overview
********

This sample demonstrates SAF boot configuration features on MEC15-based boards.
It showcase simple app that allows to initialize MEC15xx SAF module.

Building and Running
********************

The sample can be built and executed on boards using west.


Sample output
=============

.. code-block:: console

   Boot config detected: SAF
   Initializing SPI flash device
   Configuring SAF module
   Starting SAF module
   Wait...
   Writing SPI flash via SAF
   Reading SPI flash via SAF

note:: The values shown above might differ.
